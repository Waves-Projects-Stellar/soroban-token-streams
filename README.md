### soroban-token-streams

A composable payment primitive that enables continuous, real-time token streaming on Soroban. Designed for salaries, grants, subscriptions, and DAO disbursements.

---

### ✨ Features

* Per-second token streaming
* Pause, resume, cancel streams
* Real-time withdrawable balances
* Multi-stream management
* Extensible payment logic

---

### 🧱 Module Breakdown

* **Core Engine**

  * Stream creation
  * Rate calculation
  * Balance tracking

* **Extensions**

  * Cliff periods
  * Vesting schedules
  * Milestone-based unlocks

* **Frontend**

  * Stream dashboard
  * Withdraw interface

* **Automation**

  * Auto-withdraw bots
  * Expired stream cleanup

---

### 🏗 Technical Architecture

* Contracts: Rust + Soroban SDK
* Frontend: Next.js + Tailwind
* Backend: Node.js automation scripts
